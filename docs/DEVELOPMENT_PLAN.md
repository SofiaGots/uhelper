# 📋 Полный поэтапный план разработки UHelper AI-ассистента

## Обзор проекта
**Проект:** UHelper - AI-ассистент для помощи в поступлении в университеты
**Текущее состояние:** MVP Telegram-бот с базовыми агентами
**Срок реализации:** 12 недель (84 дня)
**Целевая дата завершения:** Март 2026

### Текущее состояние проекта (MVP Telegram-бот)

**✅ Уже реализовано:**
- Базовый Telegram-бот с архитектурой AI-агентов
- Оркестрационный агент (`OrchestratorAgent`)
- Агент данных университетов (`UniversityDataAgent`)
- Агент анализа профиля (`ProfileAnalyzerAgent`)
- База данных из 10 ведущих российских университетов
- Интеграция с Claude AI (Anthropic API)

**🔧 Требует исправления:**
- Обновление AGENTS.md с указанием правильной API конфигурации

## 🔧 ТЕХНИЧЕСКИЕ СПЕЦИФИКАЦИИ

### Архитектура данных
```
База университетов (SQLite/PostgreSQL)
├── Основная информация (название, город, программы)
├── Требования к поступлению (баллы, предметы)
├── Сроки подачи документов
└── Контактная информация

Профили пользователей
├── Академические данные (оценки, тесты)
├── Предпочтения (интересы, города)
├── История взаимодействий
└── Настройки уведомлений

Сессии и контекст
├── Текущий диалог
├── Состояние пользователя
└── Временные данные
```

### API и интеграции
- **Telegram Bot API** - основной интерфейс
- **Anthropic API (Claude)** - AI обработка
- **Планируемые интеграции**: университетские API, образовательные порталы

### Стек технологий
- **Backend**: Python 3.8+, asyncio
- **AI**: Anthropic Claude API
- **База данных**: SQLite (разработка), PostgreSQL (продакшн)
- **Деплой**: Docker, cloud hosting

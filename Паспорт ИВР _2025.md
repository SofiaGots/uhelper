+--------------------------------+-------------------------------------+
| **Продукт ИВР (подробное       | Мой проект - интеллектуальный       |
| описание)**                    | ассистент-навигатор по поступлению  |
|                                | в вузы.\                            |
|                                | Он помогает ученику спланировать    |
|                                | стратегию поступления, оценить      |
|                                | шансы и не пропустить дедлайны.     |
|                                |                                     |
|                                | Основные функции:                   |
|                                |                                     |
|                                | 1)  Сбор актуальной информации о    |
|                                |     вузах, программах, сроках,      |
|                                |     стипендиях.                     |
|                                |                                     |
|                                | 2)  Анализ данных о пользователе    |
|                                |     (успеваемость, интересы,        |
|                                |     профиль).                       |
|                                |                                     |
|                                | 3)  Прогноз вероятности поступления |
|                                |     в разные университеты.          |
|                                |                                     |
|                                | 4)  Календарь дедлайнов и           |
|                                |     автоматические уведомления.     |
|                                |                                     |
|                                | 5)  Персональные рекомендации по    |
|                                |     подготовке.                     |
|                                |                                     |
|                                | Формат реализации: Telegram-бот     |
|                                | (MVP-версия) с возможностью         |
|                                | последующего переноса в             |
|                                | веб-платформу.                      |
+--------------------------------+-------------------------------------+
| **Критерии качества продукта   | 1)  Функциональность (все основные  |
| ИВР**                          |     функции (информация, календарь, |
|                                |     тесты, анализ) работают         |
|                                |     корректно)                      |
|                                |                                     |
|                                | 2)  Удобство интерфейса             |
|                                |     (пользователь может выполнить   |
|                                |     ключевые действия без           |
|                                |     затруднений)                    |
|                                |                                     |
|                                | 3)  Персонализация (ассистент       |
|                                |     учитывает данные конкретного    |
|                                |     ученика)                        |
|                                |                                     |
|                                | 4)  Актуальность информации (данные |
|                                |     о вузах и дедлайнах обновляются |
|                                |     из проверенных источников)      |
|                                |                                     |
|                                | 5)  Эффективность (пользователь     |
|                                |     улучшает понимание процесса     |
|                                |     поступления)                    |
|                                |                                     |
|                                | 6)  Отзывы (минимум 80 %            |
|                                |     пользователей оценивают         |
|                                |     продукт, как «полезный» или     |
|                                |     «очень полезный»)               |

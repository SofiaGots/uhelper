# UHelper - AI Assistant for University Admission

🚀 **UHelper** - это интеллектуальный ассистент для персонализированной поддержки на всех этапах поступления в университет. Проект разрабатывается как Telegram-бот с возможностью последующего расширения до веб-платформы.

## 🎯 О проекте

**Целевая аудитория:** Ученики 9-11 классов, готовящиеся к поступлению в вузы (Россия и за рубежом)

**Основные функции:**
- 📊 Анализ профиля студента и подбор подходящих университетов
- 📅 Управление сроками поступления и автоматические уведомления
- 📚 Подготовка к экзаменам (ЕГЭ, SAT, IELTS и др.)
- 📋 Помощь с документами и заявлениями
- 🎯 Персональные рекомендации по стратегии поступления

## 🏗️ Архитектура

Проект построен на основе системы из 8 AI-агентов:

1. **OrchestratorAgent** - Главный координатор
2. **UniversityDataAgent** - Информация о вузах
3. **ProfileAnalyzerAgent** - Анализ профиля студента
4. **TimelineAgent** - Управление сроками
5. **ExamPrepAgent** - Подготовка к экзаменам
6. **DocumentAgent** - Управление документами
7. **AssessmentAgent** - Интерактивные оценки
8. **NotificationAgent** - Уведомления

Подробная спецификация агентов доступна в [AGENTS.md](./AGENTS.md)

## 🚀 Быстрый старт

### Требования
- Python 3.8+
- Telegram Bot Token
- Anthropic API Key (для AI-агентов)

### Установка
```bash
# Клонировать репозиторий
git clone https://github.com/SofiaGots/uhelper.git
cd uhelper

# Установить зависимости
pip install -r requirements.txt

# Настроить окружение
cp .env.example .env
# Отредактировать .env с вашими ключами
```

### Запуск
```bash
python main.py
```

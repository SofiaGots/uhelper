# UHelper - Минимальная версия AI-ассистента для поступления

🚀 **UHelper** - это интеллектуальный Telegram-бот для помощи в поступлении в университеты России. Минимальная версия включает основные функции поиска университетов и анализа профиля.

## 🎯 Что умеет текущая версия

- **Поиск университетов** - информация о 10 ведущих вузах России
- **Анализ профиля** - оценка шансов поступления и рекомендации
- **Требования к поступлению** - проходные баллы и предметы ЕГЭ
- **AI-анализ** - интеллектуальная обработка запросов через Claude AI

## 🏗️ Архитектура проекта

Проект построен на модульной системе AI-агентов:

### Основные агенты:
1. **OrchestratorAgent** - Главный координатор, определяет намерения пользователя
2. **UniversityDataAgent** - Работа с базой университетов (10 ведущих вузов)
3. **ProfileAnalyzerAgent** - Анализ профиля студента и рекомендации

### Структура проекта:
```
uhelper/
├── src/
│   ├── agents/           # AI-агенты
│   │   ├── base_agent.py
│   │   ├── orchestrator.py
│   │   ├── university_data_agent.py
│   │   └── profile_analyzer_agent.py
│   ├── models/           # Модели данных
│   │   └── base.py
│   └── bot.py           # Telegram бот
├── main.py              # Главный файл запуска
├── requirements.txt     # Зависимости
├── .env.example        # Шаблон переменных окружения
└── README.md          # Эта документация
```

## 📋 Требования

- Python 3.8+
- Telegram Bot Token (получить у @BotFather)
- Anthropic API Key (для AI-функций)

## 🚀 Быстрый запуск

### 1. Установка зависимостей
```bash
pip install -r requirements.txt
```

### 2. Настройка окружения
```bash
cp .env.example .env
```

Отредактируйте файл `.env`:
```env
TELEGRAM_BOT_TOKEN=ваш_токен_бота
ANTHROPIC_API_KEY=ваш_ключ_anthropic
```

### 3. Запуск бота
```bash
python main.py
```

## 💬 Как использовать

После запуска бота в Telegram:

### Основные команды:
- `/start` - Начало работы
- `/help` - Справка
- `/profile` - Инструкция по анализу профиля
- `/universities` - Поиск университетов

### Примеры запросов:
```
«Какие IT-университеты в Москве?»
«Проанализируй мой профиль: у меня 5 по математике, 4 по физике»
«Какие требования для поступления в МГУ?»
«Программы экономики в Санкт-Петербурге»
```
